# Project Memory

## Current Scope

Local Streamlit MVP for multi-file project update ingestion, PMBOK-aligned classification, and Teach the App learning.

## Included

- Upload CSV, Excel, Word, PowerPoint, text, and Markdown files.
- Normalize records into source metadata plus update text.
- Classify risks, blockers, dependencies, decisions, data issues, compliance risks, adoption risks, scope changes, overdue actions, and general updates.
- Save local update logs and user corrections.

## Excluded

- Gmail
- Jira
- OAuth
- Database
- OCR
- External AI APIs
- Cloud deployment automation
